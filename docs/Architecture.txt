Notes:
  - [ AWS WAF ] Security layer protecting CloudFront with Bot Control, Rate Limiting, and IP Reputation.
  - [ Load Balancer ] is a single ALB (aws_lb.main) spanning both public subnets for HA.
  - [ Regional NAT GW ] is a single logical resource in Automatic Mode that AWS scales dynamically.
  - [ Odoo Service ] Odoo connects to EFS Mounting target for file storage and PgBouncer for database pooling.
  - [ PgBouncer HA ] It uses ECS Service Connect (pgbouncer.odoo.local) for discovery.
  - [ EFS Mount Target ] are two mount targets of one shared EFS file system (aws_efs_file_system.odoo).
  - [ RDS Postgres ] is a single instance with multi_az=true; AWS manages the standby replica automatically.

Additional Infrastructure Notes (AI & Caching):
  - [ Amazon Bedrock ] Supervisor Agent (Claude 3.5 Sonnet) orchestrating autonomous research workflows.
  - [ Knowledge Base ] RAG implementation using OpenSearch Serverless as a vector database.
  - [ Research Lambdas ] Functions ("Librarian" & "Odoo Integrator") that execute research and sync data.
